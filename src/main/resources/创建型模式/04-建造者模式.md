# 建造者模式

## 介绍

建造者模式是创建型模式之一，使用多个简单的对象创建一个复杂的对象，定义不同的构造类使得相同的构建过程可以创建不同的对象

## 模块

1. Builder：构造者接口
2. ConcreteBuilder：具体构造者，实现构造器接口，定义复杂对象的创建流程
3. Product：被创建的对象
4. Director：指导者，调用具体构造者创建对象的各个部分
